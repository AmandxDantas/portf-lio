
Look4Nick é uma ferramenta avançada de OSINT (Open Source Intelligence) projetada para a identificação e mapeamento de presenças digitais através de nomes de usuário. Com uma interface de alto contraste e filtros inteligentes, o Look4Nick transforma a busca de perfis em uma experiência ágil, visual e estratégica.

Identidade Visual
O projeto adota uma estética Cyber-Noir, priorizando a legibilidade em ambientes de investigação e o conforto visual em longas sessões de uso.

Fundo Principal: Preto Carvão (#121212) para profundidade e redução de fadiga ocular.

Destaques e Ações: Roxo Neon (#8F00FF) para guiar o olhar do usuário para botões de busca e links ativos.

Alertas e Status: Variações de Roxo Magenta e Branco Gelo para feedbacks de sistema.

2. Exportação Ágil (One-Click Copy)
Otimização do fluxo de trabalho investigativo através de um botão dedicado para copiar todas as URLs encontradas diretamente para a área de transferência, facilitando a criação de relatórios externos.

3. Histórico de Sessão Volátil
Mantenha o controle das suas últimas buscas sem comprometer a privacidade. Uma barra lateral persistente permite alternar entre os nomes de usuário pesquisados durante a sessão atual para comparação rápida de resultados.

4. Modo de Exibição de Inexistência
Transparência total no processo de busca. O sistema exibe não apenas onde o alvo foi encontrado, mas também confirma os locais onde o nome de usuário está disponível, ajudando a descartar hipóteses ou reservar identidades.

 Tecnologias Utilizadas
Frontend: HTML5, CSS3 (Custom Properties para a paleta Dark/Purple).

Lógica: JavaScript Assíncrono para requisições em tempo real.

Arquitetura: Decomposição lógica para alta performance de varredura.
